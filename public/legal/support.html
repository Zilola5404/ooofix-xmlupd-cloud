Генерация XML (УПД)

Приложение для 1С-Битрикс24: автоматическое формирование XML универсального передаточного документа (УПД) по данным сделки CRM, сохранение на Диск и в поле «Файл УПД» для выгрузки в 1С и ЭДО.

Возможности приложения

Кнопка «Сформировать УПД» в карточке сделки CRM
Робот CRM для автоматической генерации XML
Проверка XML по XSD-схеме ФНС
Сохранение в папку XML на общем Диске портала
Запись файла в поле «Файл УПД» сделки
Настройки реквизитов продавца и подписанта
