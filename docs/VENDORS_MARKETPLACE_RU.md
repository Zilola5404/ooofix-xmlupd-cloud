## Полное описание

```
Приложение «Генерация XML (УПД)» автоматизирует подготовку универсального передаточного документа (УПД) в формате XML по данным сделки Bitrix24 CRM.

Для кого: бухгалтерия, финансовые отделы, интеграторы 1С и ЭДО, компании, которые выставляют закрывающие документы из CRM.

Возможности:
• формирование XML УПД по товарным позициям, суммам и реквизитам продавца и покупателя из сделки;
• проверка XML по актуальной XSD-схеме ФНС;
• сохранение файла в папку XML на общем Диске портала;
• запись файла в пользовательское поле сделки «Файл УПД» для дальнейшей выгрузки в 1С;
• кнопка «Сформировать УПД» в карточке сделки;
• робот CRM и сценарии автоматизации;
• настройки реквизитов организации, подписанта, версии формата и кодировки;
• журнал сформированных документов и диагностика REST-прав.

При установке создаются пользовательские поля «Номер УПД (1С)» и «Файл УПД» для сделок и смарт-процесса «Счета».

Приложение работает в облаке Bitrix24, не требует установки на сервер клиента. Данные обрабатываются через REST API в объёме, необходимом для генерации документов.

Разработчик: ООО «РЕШЕНИЕ». Сайт: https://ooofix.ru
```

## Описание процесса установки

> Без контактов (email, телефон, сайт). Бренд портала — только **«Битрикс24»** (кириллица), не «Bitrix24».

```
1. В каталоге приложений нажмите «Установить» и подтвердите запрашиваемые права (CRM, Диск, бизнес-процессы, пользовательские поля).

2. После установки откройте приложение в левом меню «Генерация XML (УПД)».

3. На странице настроек заполните реквизиты продавца (ИНН, КПП, наименование, адрес), данные подписанта, версию формата XML и кодировку. Нажмите «Сохранить».

4. При первой установке автоматически создаются поля «Номер УПД (1С)» и «Файл УПД» для сделок. Если поля уже были созданы ранее, нажмите «Создать UF-поля» в настройках.

5. Откройте сделку с заполненными товарными позициями и контрагентом. В карточке сделки нажмите «Сформировать УПД».

6. Готовый файл сохранится в папке XML на общем Диске и в поле «Файл УПД» сделки для передачи в 1С или систему ЭДО.

7. Для автоматической генерации добавьте робот «Сформировать XML УПД» в сценарии автоматизации CRM.

Требования: облачный портал с модулем CRM, права администратора для установки, заполненные реквизиты компании и товарные позиции в сделке.
```
